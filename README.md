# Spelling Bee Game & Solver (C)

## Overview
This project is a C implementation of a Spelling Bee-style word puzzle game and solver. It reads a dictionary file, generates or accepts a custom letter hive, and finds all valid words that can be formed using the hive rules.

The program supports both interactive gameplay and automated solving using brute force or optimized recursive search.

---

## Features

- 📚 Loads words from a dictionary file using file I/O
- 🔤 Builds a letter hive with unique, alphabetized characters
- 🎮 Play mode where users submit valid words
- 🤖 Solver mode (brute force or recursive backtracking)
- ⭐ Pangram and perfect pangram detection
- 🧠 Valid word checking using hive constraints
- 📊 Scoring system similar to NYT Spelling Bee
- ⚙️ Command-line configurable settings
